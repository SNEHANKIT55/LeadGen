# LeadGen AI – Business Lead Generator

## Overview

LeadGen AI is a simple MVP tool that generates business leads based on a product or service keyword.

Users can enter a keyword and the system generates business contacts such as company name, phone number, email, and location.

## Features

* Keyword-based lead generation
* Table view of business leads
* Export leads to CSV
* Copy phone number and email
* Clean SaaS-style UI

## Tech Stack

Frontend: React + TailwindCSS
Build Tool: Vite
Language: TypeScript

## How to Run

1. Install dependencies
   npm install

2. Run the project
   npm run dev

3. Open in browser
   http://localhost:5173

## Example Keywords

* Solar Panel Dealers
* Water Purifier Suppliers
* TDS Meter Distributors
